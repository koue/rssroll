# rssroll
RSS aggregator written in C
version 0.7

RSS crawler which stores feeds into sqlite3 database.
Simple web interface to browse them grouped by category.

Used libraries:
curl
libcez
libxml2
sqlite3

Read INSTALL and UPDATING files for more information.
